Calendar

Each project has its own calendar. Use it to track build days, performance dates, inspections, deliveries, and any other time-bound commitment tied to the project. Events feed the Upcoming Events widget in Stage 3 Production and the Next 10 Days panel on your Dashboard.

📷
Screenshot: The project calendar showing a month grid with color-coded event pills in various day cells

Opening the Calendar

The calendar is accessible from the top bar inside any project. Click the calendar icon to open it. It is a per-project view scoped entirely to the current project.

Navigating the Calendar

The calendar displays a standard month grid, Sunday through Saturday. Today's cell is highlighted. Use the ‹ and › buttons on either side of the month label to move back and forward one month at a time.

Event Types

When you create an event you assign it a type. Each type renders as a color-coded pill in the calendar cell.

Type Color
Rehearsal Blue
Performance Purple
Meeting Slate
Inspection Orange
Delivery Green
Deadline Red
Build Day Amber
Custom Gray

Adding Events

Click any day cell to open the new event form. A + button also appears in the top-right corner of each cell on hover and triggers the same form.

Field Notes
Title Required. Press Enter to submit.
Type Dropdown. Determines the event pill color.
Time Optional. Auto-fills to the current hour on focus.
Notes Optional free text.

Click Add Event or press Enter in the title field to save. The event appears as a color pill in the correct day cell immediately.

Right-clicking any day cell opens a context menu with a single option: + Add Event. This opens the same form pre-filled with that date.

Editing and Deleting Events

Click any event pill to open the detail panel. All fields are editable including the date, so you can reschedule by changing the date field directly. Click Save to confirm changes.

To delete click Delete in the panel footer. A confirmation prompt replaces the buttons. Confirm to remove.

Right-clicking an event pill opens a context menu with Edit (opens the detail panel) and Delete (skips to the confirmation step inside the panel).

Events do not repeat. There is no recurring event option. For events that happen on multiple dates add them individually or add a note in the event description referencing the full schedule.

How Events Surface Elsewhere

Next 10 Days on Dashboard — Events across all your projects that fall within the next 10 days appear in the Next 10 Days panel on the Dashboard. Toggle this panel on or off in Account Settings → Project Defaults.

Upcoming Events widget in Stage 3 — Events for the current project in the next 14 days appear in the Upcoming Events widget on the Production board. A scope toggle in the widget switches between this project only and all projects.

Adding events from the Dashboard — You can add events without opening a project. Click + Add Event in the Next 10 Days panel on the Dashboard, select a project from the dropdown, and fill in the event details.
