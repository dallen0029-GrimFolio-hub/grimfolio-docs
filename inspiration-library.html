Inspiration Library

The Inspiration Library is where reference images live. Upload production stills, mood board pulls, concept sketches, floor plan photos — anything that belongs to your creative mental model. Images in the library can be linked to individual cards, dropped onto the canvas as background images, or used as the starting point for GrimVision concept art generation.

📷
Screenshot: Inspiration Library panel open in Build showing the image grid, upload zone, and tag filter bar

Opening the Library

The Inspiration Library is accessible two ways. From the Dashboard, click the Inspiration Library card in the right column to open the full standalone library page. From inside a project in Stage 2, click the Libraries button in the left toolbar and select Inspiration Library to open it as a panel on the right side of the canvas.

Uploading Images

The upload zone is at the top of the panel or page. Drag an image onto it or click to open a file picker.

Detail Spec
Accepted formats JPEG, PNG, WebP, GIF
Maximum file size 10MB per image
Free plan limit 50 images total
Pro plan limit 500 images total

The image count is shown in the upload zone as you approach your plan limit. When you reach the ceiling the zone is disabled and shows a message. Images appear in the grid immediately after uploading.

Managing Images

Each image tile shows a thumbnail, filename, and any tags applied to it.

Action How
Rename Click the filename below the thumbnail. Type a new name and press Enter or click away.
Add a tag Hover the tile, click + tag in the tag row. Type the tag and press Enter.
Remove a tag Hover the tile. An × appears next to each tag. Click it.
Delete image Hover the tile, click the ✕ button in the overlay. Deletes immediately.
Link to a card Hover the tile, click ⊕ Link. See below.

Tags are lowercase and stored per image. Images can have multiple tags. A tag filter bar appears above the grid once any image has been tagged. Click a tag to filter the grid. Click All or the active tag again to clear the filter.

A small purple badge on a tile shows how many cards that image is currently linked to.

Linking Images to Cards

There are two ways to link an image to a card.

From the library panel — Hover the image tile and click ⊕ Link. A hint bar appears: "Click a node on the canvas to link this image." Click any card. The link is created and the image appears in the card's Inspiration section in the detail drawer. Press Escape to cancel.

From the card drawer — Open a card by double-clicking it. In the Inspiration section click + Link. An inline image picker opens showing all your library images. Click one to link it. Already-linked images show a purple checkmark and cannot be selected again.

To unlink an image from a card open the card drawer, go to the Inspiration section, and click the ✕ button on any linked thumbnail.

GrimVision

The GrimVision button appears in the Inspiration section of any card's detail drawer alongside your current credit balance.

Clicking it opens the GrimVision panel. Select a niche — Haunted House, Escape Room, Film, Cosplay, and others — then a style template suited to your project type. Add a visual description of what you want to see and generate. Each generation uses credits.

Generated images are saved to your GrimVision Library and appear as thumbnails in the card's Inspiration section. They are also available in the canvas image picker's GrimVision Library tab so you can place them directly on the canvas as background images.

For full GrimVision documentation see the GrimVision page.

Inspiration in Stage 3

The Inspiration Board widget in Stage 3 Production surfaces all inspiration images linked to cards across the project in a single read-only grid. Each thumbnail shows the image, an optional title, and the card it belongs to. Click any thumbnail to open it in a lightbox.

The board is read-only. Manage images in Stage 2 via the Inspiration section of each card's detail drawer.
